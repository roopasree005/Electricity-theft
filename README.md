# Anomaly Detection Project 

This project uses the Isolation Forest algorithm to detect anomalies in a dataset.

## What this project does:
- Loads dataset
- Selects numeric columns
- Scales features using StandardScaler
- Trains Isolation Forest model
- Predicts anomalies (Normal or Abnormal)
- Generates anomaly scores
- Visualizes anomaly scores
- Saves results and model files

## Files included:
- notebook.ipynb → Complete code and results
- scaler.pkl → Saved StandardScaler model
- isolation_forest.pkl → Saved Isolation Forest model
- anomaly_results.csv → Predictions + anomaly scores
- 2-week progress files → Previous work
- README.md → Project explanation

## How to run:
1. Open `notebook.ipynb`
2. Run all cells
3. Check anomalies and scores
